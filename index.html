Journaly

Your personalized journaling journey

Project Description

The goal of our application is to allow users to have private journals and receive automated feedback on their entries in order to enhance the journaling process and better improve their wellbeing. It is not intended to replace therapy or other resources, but rather to act as an additional tool- the about page of the site lists some crisis-related resources for those who may need them.

Implementation

We used a MERN stack for this project- MongoDB, Express, React, and Node.js. It is deployed on Heroku.

Other resources used were:
  • Bootstrap as our front-end library
  • plotly.js, an open-source graphing library
  • react-d3-cloud, a word cloud react component
  • IBM Watson NLU for text sentiment analysis
